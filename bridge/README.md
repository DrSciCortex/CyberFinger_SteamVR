# CyberFinger Bridge

GUI application that connects CyberFinger BLE controllers to your PC
in two modes:

- **VR Mode** — forwards BLE input via UDP to the CyberFinger SteamVR driver
- **Gamepad Mode** — combines left + right controllers into a virtual Xbox 360 gamepad

## Quick Start (from source)

```
pip install -r requirements.txt
python cyberfinger_gui.py
```

## Building the Installer

### Prerequisites

1. Install build dependencies:
   ```
   pip install pyinstaller pillow
   pip install -r requirements.txt
   ```

2. Install [Inno Setup 6](https://jrsoftware.org/isdl.php) (for creating the installer)

3. (Optional) Download [ViGEmBus driver installer](https://github.com/nefarius/ViGEmBus/releases)
   and place it as `installer/ViGEmBus_Setup.exe` — the installer will offer to
   install it automatically for Gamepad mode users.

### Build Steps

```
build.bat
```

This will:
1. Convert the PNG icon to ICO format
2. Run PyInstaller to create `dist/CyberFingerBridge.exe`
3. Prepare the distribution files

To create the Windows installer:
```
cd installer
iscc setup.iss
```

The installer will be created at `dist/installer/CyberFingerBridge_Setup_1.0.0.exe`.

## Gamepad Button Mapping

| Bit  | Button       | Right Hand             | Left Hand              | XInput btn # |
|------|--------------|------------------------|------------------------|--------------|
| 0x01 | Trigger      | A                      | X                      | 1 / 3        |
| 0x02 | Grip         | B                      | Y                      | 2 / 4        |
| 0x20 | Menu         | RB (R-Shoulder)        | LB (L-Shoulder)        | 6 / 5        |
| 0x40 | Joy Click    | R3 (R-Thumb)           | L3 (L-Thumb)           | 10 / 9       |
| 0x80 | Start/Select | Start                  | Back                   | 8 / 7        |
| 0x04 | C            | btn 11 (D-pad Up raw)  | btn 14 (D-pad Right raw) | 11 / 14    |
| 0x08 | D            | btn 12 (D-pad Down raw)| btn 15 (Guide raw)     | 12 / 15      |
| 0x10 | E            | btn 13 (D-pad Left raw)| btn 16 (reserved raw)  | 13 / 16      |
| —    | Joystick     | Right Stick (Y inv)    | Left Stick (Y inv)     | —            |
| —    | Trigger analog | Right Trigger        | Left Trigger           | —            |

> **Note on buttons 11–16:** The Xbox 360 `wButtons` field has unused/reserved bits
> that ViGEm passes through to DirectInput as extra buttons. C/D/E are mapped there
> directly via `gp.report.wButtons` since they have no named `XUSB_BUTTON` enum entry.
